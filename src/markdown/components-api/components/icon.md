# icon

The icon component. 

All of the possible values for the icon come from the material design icons.

## Examples

### Simple icon

```json
{
    "type": "icon",
    "value": "abc"
}
```

### Icon color & size

```json
{
    "type": "icon",
    "value": "abc",
    "color": 0xFF000000,
    "size": 100
}
```
